Design Language Summary

  • Base: Near-black (#060610) with subtle blue undertone
  • Primary accent: Purple (#8b5cf6)
  • Source colors: Spotify green, YouTube red — used only for subtle indicators
  • Glass: backdrop-filter blur(20-40px) with rgba(255,255,255, 0.04-0.08)
  • Fonts: Space Grotesk (display/numbers) + Inter (body)
  • Depth: Gradient mesh backgrounds, glow effects, layered glass surfaces
